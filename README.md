### 1)What is the difference between var, let, and const?

ANSWER: In JavaScript 'var', 'let', and 'const' differ mainly in scope and ability to re-assign. 'var' is global and hoisted. 'let' and 'const' are block-scoped (scoped to {}), with 'let' allowing reassignment and 'const' prohibiting it.

### 2)What is the spread operator (...)?

ANSWER: The Spread Operator (...), introduced in ES6, is a JavaScript syntax that expands arrays, strings or objects into individual elements or properties.

### 3)What is the difference between map(), filter(), and forEach()?

ANSWER: The primary difference between 'map()', 'filter()', and 'forEach()' lies in their return values and their intended purpose for data transformation. While all three methods work over an array and execute a provided function for each element but they handle the results differently.'map()' returns a new array of same length, 'filter()' returns a shorter array and 'forEach()' returns nothing.

### 4)What is an arrow function?

ANSWER: Arrow functions are a concise, ES6-introduced syntax in JavaScript for writing function expressions using a "fat arrow" (=>). They allow for shorter, more readable code, often omitting the function and return keywords.

### 5)What are template literals?

ANSWER: Template literals are JavaScript string literals, introduced in ES6, that allow embedded expressions, multi-line strings, and string writing using backticks (`) instead of quotes. They enhance readability by allowing developers to insert variables and expressions directly into strings using '${expression}' syntax, avoiding complex concatenation.
